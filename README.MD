<h1>CRON REMAINDER</h1>


 <p>A simple Node.js application that allows users to create events and sends users email when an event is due</p>
 <h1>Built with</h1>
 <ul>
 <li>Express</li>
  <li>Mongoose</li>
   <li>Node-cron</li>
   <li>Sendgrid</li>
 </ul>


 <h1>BStartig App</h1>
<p>Create an env file and set </p>
<p>
initialise 
MONGO_URI ,
verifiedSender and 
SENDGRID_API_KEY 
</p>
<p>Run npm start</p>
